Add User without a Home Directory
sudo useradd johnDoe

Add User with a Home Directory
sudo useradd -m janeDoe

Create user with Home Directory, Default Shell
sudo useradd chandler_bing -m -s /bin/bash

Create user with Home Directory, Default Shell and by default sudo privileges

sudo useradd testUser -m -s /bin/bash -G sudo

Delete User alongwith their Home Directory:
sudo userdel -r janeDoe
sudo userdel -r johnDoe
