Linux Compress & Expand:

tar -czvf zipFile.tgz fileToZip
tar -xzvf zipFile.tgz

; lrzip best Compression keeps original
sudo apt-get install lrzip
lrzip -z fileToZip
lrunzip words.txt.lrz
